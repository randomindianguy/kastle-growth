# Kastle — Growth Engine

An interactive tool exploring Kastle AI's market opportunity, prospect economics, and competitive positioning in mortgage AI.

Built as part of a Founder Intern application.

## What's inside

- **Prospect ROI Calculator** — Inputs for portfolio size, call volume, cost-per-call, and delinquency rate. Outputs projected savings using MBA 2024 servicing cost data and Kastle's published metrics.
- **Positioning & Moats** — Competitive analysis across the buyer's journey (Sinfield), positioning breakdown (Dunford), and defensibility mapping (Helmer's 7 Powers).
- **My First 30 Days** — Outcome-driven shipping plan: sales enablement → onboarding acceleration → proof-of-ROI loop.
